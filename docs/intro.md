# Tutorial Intro

## Getting Started

### Push code to the GitHub

```shell
GIT_USER=<gitHub Username> USE_SSH=true npm run deploy
```

### Start your site

Run the development server:

```bash
cd my-website
npm run start
```

## Install math functions

```sh
npm install --save remark-math@3 rehype-katex@5 hast-util-is-element@1.1.0
```
